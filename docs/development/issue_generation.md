# Historical Issue Generation

This page is retained only as historical context.

The old roadmap issue generator was based on an early 54-issue roadmap snapshot
with sections such as Security Remediation Sprint, Milestones 4-11, and
Continuous Improvements. That structure no longer matches the current Track A-G
roadmap, current milestone history, or current GitHub backlog.

Do not run the archived generated issue scripts as part of normal development.
They may create obsolete or duplicate GitHub issues.

Current backlog work should start from:

- [Product Roadmap](roadmap.md)
- [Milestones](../milestones.md)
- [Ecosystem Integration Matrix](../reference/ecosystem-integrations.md)
- The live GitHub issue list

If bulk issue generation is needed again, regenerate it from the current roadmap
tracks and milestone structure instead of reusing the archived snapshot in
`generated_issues/`.
