# Historical Generated Issues

This directory is intentionally archived.

DMARQ previously generated a 54-issue GitHub import from an older roadmap
snapshot. That snapshot no longer matches the current Track A-G roadmap,
milestone history, or live GitHub backlog, so the generated JSON, preview, and
shell import script were removed to prevent accidental duplicate or obsolete
issue creation.

Current backlog work should start from:

- [Product Roadmap](../roadmap.md)
- [Milestones](../../milestones.md)
- [Ecosystem Integration Matrix](../../reference/ecosystem-integrations.md)
- The live GitHub issue list

If bulk issue generation is needed again, regenerate it from the current roadmap
tracks and milestone structure.
